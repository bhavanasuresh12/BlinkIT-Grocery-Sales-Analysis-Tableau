# BlinkIT Grocery Sales Analysis Dashboard

## Project Overview

This project analyzes BlinkIT grocery sales data using Tableau to identify sales trends, outlet performance, product categories, and customer behavior.

## Tools Used

- Tableau
- Microsoft Excel
- CSV Dataset

## Dataset

- 8,523 records
- 12 columns

## Dashboard Features

- KPI Cards
- Sales by Item Type
- Fat Content Analysis
- Outlet Performance
- Sales Trend
- Interactive Filters

## KPIs

- Total Sales: 1.20M
- Average Sales: 141
- Number of Items: 8,523
- Average Rating: 3.97

## Key Insights

- Fruits & Vegetables generated the highest sales.
- Snack Foods ranked second.
- Low Fat products contributed 64.6% of sales.
- Supermarket Type 1 achieved the highest sales.
- Sales peaked for outlets established around 2018.
